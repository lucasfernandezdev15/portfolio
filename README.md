# Lucas Fernandez — Portfolio

Personal portfolio built with React. Features:

- 🖱️ **Magnetic custom cursor** with smooth lag effect
- ✨ **Scroll reveal animations** via IntersectionObserver
- 🃏 **3D tilt cards** on mouse movement  
- ⚡ **Glitch text effect** (CSS only, zero deps)
- ⌨️ **Typewriter** with rotating role strings
- 📊 **Animated skill bars** triggered on scroll
- 🎨 Dark editorial aesthetic — Syne + DM Mono

## Stack
React · CSS (no external animation libs) · CRA · Vercel

## Run locally
```bash
npm install
npm start
```

## Deploy to Vercel
Connect the repo — Vercel auto-detects CRA and builds it.
